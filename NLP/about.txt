This project is to practice the steps involved in analysing text for semantic analysis using supervised and unsupervised methods.

Supervised
----------
Classification

Unsupervised
-----------
clustering

Pre processing will include:
===========================
- removing punctuation and symbols
- lower case all text
- tokenize values i.e. make a list of the document words
- remove stopwords
- vectorize

Python libraries attempted:
--------------------------
- scikit learn
- nltk
- gensim
